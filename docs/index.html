Simter colgroup vue component.

Fork me on Github

The target is to config the table column width by colgroup/col tag.

// html
<table border="1" style="border-collapse:collapse">
  <colgroup is="st-colgroup" :columns="columns"></colgroup>
  <thead>
    <tr>
      <th v-for="(c, index) in columns">Column{{index}}</th>
    </tr>
  </thead>
</table>

Example 1: Simple Width Array

// config
columns = ["50px", "100px", "150px"]

// Result
Column{{index}}

Attention: To make td has ellipsis text, need to set `table-layout:fixed` and a suitable width on the table element, as well as `overflow:hidden;white-space:nowrap;text-overflow:ellipsis` on the table cells.

Example 2: {width: xxx} Array

// config
columns = [
  { width: "50px"  },
  { width: "100px" },
  { width: "150px" }
]
Will be flatten to ["50px", "100px", "150px"].

// Result
Column{{index}}

Attention: To make td has auto return text, just set `word-break:break-all` on the table cells.

Example 3: Nested Array

// config
columns = [
  "50px",
  { width: "100px" },
  {
    children: [
      { width: "150px" },
      "60px"
    ]
  }
]
Will be flatten to ["50px", "100px", "150px", "60px"].
